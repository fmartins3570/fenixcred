# Documentacao do Projeto

Este diretorio concentra os guias operacionais e tecnicos que antes estavam soltos na raiz.

## Estrutura

- `accessibilidade/`: praticas de contraste e acessibilidade.
- `deploy/`: instrucoes de deploy, hostinger e configuracoes de `.htaccess`.
- `integracoes/`: integracoes externas (Google Reviews, Analytics, logo).
- `performance/`: otimizacoes de Core Web Vitals, bundle, CSS/JS e imagens.
- `seo/`: guias de SEO, sitemap, breadcrumbs e links internos.
- `projeto/`: registros gerais de melhorias implementadas.

## Observacao

Arquivos essenciais de execucao continuam na raiz:

- `package.json`
- `vite.config.js`
- `index.html`
- `.htaccess`
